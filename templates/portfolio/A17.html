This website

This time, instead of adding more HTML and JS clutter to this website, i decided to build out a small backend for it.

Since I have my own infrastructure it is possible to use whatever language for that. The server infrastructure provided by the university only allows php to my knowledge and then it is mostly just enough to run Wordpress. I chose Rust for this task.

Rust was one of the backend choices of all time, since it is slightly slower to write and is quite challenging in it's own respect. But, this is offset by the safety it brings. And because the website is exposed to the internet and will have a database of storing important stuff, this safety is a godsend.

So far I enjoy the experience.

Still in development to this day.
