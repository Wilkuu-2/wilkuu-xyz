Remote code for controlling a threaded robot.

Photo and robot construction by B. Wever

This was made for our Graduation Project (PWS). Me and a friend of mine, who is currently enrolled at Saxion in Enchede decided to make a robot. I was in charge to write the code to control the robot. As we used a raspberry pi I did't have to write really low level stuff, so I went with python with this project, which backfired extremely hard when it came to actually sending video. Rewriting the code with my newfound experience in C++ is one of the things on my TODO list. The robot is not fully tested yet but this is subject to change.

Project was on-going from late 2020 till march of 2021
