Arduino controlled additive synthesizer

For this project I used a arduino to create an input device (I know it's barely visible). Using this arduino contraption I controlled a synthesizer that adds waves together to create a sound. Most of the sounds you can create with it sound horrendous, but others sound quite funky. You can also see all the waves graphically, including the final combined wave. This is where the program is the most interesting. If you can replicate the arduino device, you can give it a try.
