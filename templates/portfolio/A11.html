A GUI library for Processing

After programming in processing for almost a year, I had a lot of ideas to extend the capability of Processing, because of how primitive the tools processing gives you are. This resulted in a concept and later a really early implementation of a GUI framework for Processing 4 based loosly on QT. There will be more to show later, after the academic year ends. But the frawework is already published on Github. So go check it out.
This project is on-going but it is temporarily shelved at the time of writing
